# KeenKeeper

KeenKeeper is a personal relationship management app that helps you stay connected with the people who matter most. It reminds you when to reach out, tracks your interactions, and shows you analytics about your friendships.

## Technologies Used

- Next.js 16
- React
- Tailwind CSS
- Recharts
- Lucide React
- React Hot Toast

## Key Features

- **Friend Dashboard** — See all your friends at a glance with their contact status (on-track, almost due, overdue)
- **Interaction Tracker** — Log calls, texts, and video chats directly from a friend's profile and view the full history in the Timeline
- **Friendship Analytics** — Visual donut chart showing your interaction breakdown by type so you can see how you stay in touch
